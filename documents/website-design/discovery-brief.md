# Discovery brief

The intake for a website design. Copy this file per client to `~/Downloads/lavder-discovery-<client>-YYYY-MM-DD.md`, fill it during/after the kickoff call, and design from it. The goal is to leave discovery with **no guesses left**: tier, track, brand layer, sitemap seed, and content reality all captured.

Fields marked **(blocks design)** must be answered before visual work starts. If a client can't answer one, that is itself a finding to surface.
