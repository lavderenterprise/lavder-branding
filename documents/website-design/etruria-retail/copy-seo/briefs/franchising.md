## Content brief : franchising

Worked example of the Step 4 per-page content brief (playbook.md, section B). The outline, H1, slug and META below match the shipped `franchising.html` exactly: this brief is the contract that page was written against.

SEARCH INTENT     : transactional + the SERP rewards a landing page (offer + proof + form), audience = shop owners and entrepreneurs in central Italy evaluating an affiliation, knowledge level = business-aware but new to this group. Reader wants to know what they get, how it works, and how to apply.

PRIMARY KEYWORD   : franchising supermercato
SECONDARY         : aprire supermercato in franchising, affiliazione supermercato, insegna in franchising, convertire negozio

PAGE TYPE         : landing

WINNING OUTLINE   :
  H1   : "Porta Etruria nella tua città." (carries the brand entity + the transactional promise; primary keyword reinforced in the intro and META)
  slug : /franchising.html
  - Hero / intro [FS-intent] : one-paragraph answer above the fold ("Sei un negoziante o un imprenditore? Con un'insegna del gruppo hai la forza di una centrale e la libertà di restare te stesso.") + two CTAs (Candidati ora -> #candidati, Perché con noi -> #perche). Keyword: affiliazione supermercato.
  - H2 "Perché Etruria" (#perche) : the four reasons (rete da 132 negozi, marca propria Sapori di Etruria, clienti fedeli con Sempre con te, libertà di gestione). Format: value grid. Keyword: insegna in franchising.
  - H2 "Cosa ti diamo" : the six concrete deliverables (insegna e format, logistica, marketing, formazione, software, affiancamento). Format: card list [FS-list]. Keyword: cosa offre il franchising supermercato.
  - H2 "Il percorso" : the four steps (ci conosciamo, il progetto, apertura, crescita). Format: numbered path. Keyword: come aprire un supermercato in franchising.
  - H2 "Parliamone." (#candidati) : the application form + the promise "il nostro team sviluppo rete ti ricontatta entro due giorni lavorativi, senza impegno." Keyword: candidatura franchising.

META              :
  title       : "Diventa franchisee · Etruria Retail"
  description : "Apri o converti il tuo negozio con un'insegna del gruppo Etruria: insegna e format, logistica, marca propria, marketing, formazione e una rete che fa squadra."

INFORMATION GAIN  : the named, itemised exchange of this specific group, not a generic franchise pitch. The page states exactly what the franchisee gets (six deliverables) AND what stays theirs (gestione e rapporto col cliente), plus the real four-step path with a concrete promise (ricontatto entro due giorni lavorativi) and the real network size (132 negozi). A competitor's "contattaci per informazioni" page does not commit to this.

E-E-A-T           : the real network scale (132 punti vendita, 1962 founding visible site-wide), the named own-brand (Sapori di Etruria), the named loyalty programme (Sempre con te) as a traffic proof, the named internal function ("team sviluppo rete" / "area manager dedicato") that owns the relationship, and a dated, bounded commitment (two working days). Trust: the contact page and the group address (Via dell'Etruria 1, 58100 Grosseto) back the legal entity.

INTERNAL LINKS    :
  - franchising -> insegne.html : anchor "una delle insegne Etruria" (spoke -> pillar; lets the reader pick the format before applying).
  - franchising -> sempre-con-te.html : anchor "la carta Sempre con te" (supports the "clienti fedeli" claim with the real programme).
  - franchising -> punti-vendita.html : anchor "una rete da 132 negozi" (proves the network claim with the live locator).
  - franchising -> contatti.html : anchor "team sviluppo rete" (gives a second contact route beyond the form).
  - chi-siamo.html -> franchising : inbound anchor "Diventa franchisee" / "Lavora con noi" (pillar -> spoke; no orphan).
